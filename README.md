## 📖 About This Repository

Welcome to my **Applied Machine Learning Lab**. This repository documents my journey and experiments in the field of Artificial Intelligence, moving from foundational statistical learning to advanced Applied AI systems.

Unlike standard code repositories, the notebooks here are designed as **research documents**. They include:
* **Mathematical Foundations:** Explanations of the algorithms (e.g., Euclidean distance, Vector spaces).
* **Data Visualization:** In-depth Exploratory Data Analysis (EDA) to understand the underlying distributions.
* **Implementation:** Clean, documented Python code using industry-standard libraries.

## 🚀 Getting Started

To explore these notebooks locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/applied-ml-lab.git](https://github.com/your-username/applied-ml-lab.git)
    cd applied-ml-lab
    ```

2.  **Install dependencies:**
    ```bash
    pip install numpy pandas matplotlib scikit-learn mglearn
    ```

3.  **Launch Jupyter:**
    ```bash
    jupyter notebook
    ```
